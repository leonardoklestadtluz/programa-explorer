* O que é programação?
R: É o ato de entender um problema do mundo real e abstrair informações deste problema. Após, organizar estas informações de forma lógica para transforma-las em um algoritmo através de uma linguagem de programação de sua preferência. Assim o computador entenderá as instruções escritas na linguegem e executará este algoritmo em sequência até a última linha de instrução.

Exemplo: fazer um bolo, andar de bicicleta, viajar, acordar, dormir e tantas outras situações do dia-a-dia.

==========================================

* Lógica de programação?
R: É a forma que ordenamos nossos pensamentos para transforma-los em instruções e são estas intruções que são passadas para o computador.

O computador recebe nossas instruções e entende elas como ordens. E processa estas ordens passadas para ele através da linguagem de programação, com isso aquele algoritmo é transaformado em um programa de computador.

* O que é e como funciona a web?

São várias redes de computadores distribuídas pelo planeta inteiro e que estão interligadas. Esta rede funciona da seguinte forma: abrimos o navegador de internet e digitamos um endereço, por exemplo: https://www.rocketseat.com.br . Este endereço é transformado e um número de IP que pergunta ao servidor se ele tem conteúdo deste endereço para nos retornar. Se tiver, o servidor retornará uma página estruturada e estilizada, podendo ser interativa ou não. Se não tiver conteúdo, servidor também retornará uma página estruturada e estilizada, porém com uma mensagem informando que não encontrou a página por algum motivo.

* Front-end vs Back-end

Front-end (quem faz o pedido) -> É a parte do site que o usuário enxerga no navegador. Composta por HTML (estrutura), CSS (estilo) e Javascript (animações).

Back-end (quem recebe o pedido) -> Cuida onde os dados estão armazenados. Contém a lógica da aplicação. Composta por Javascript (Node.JS) e banco de dados (PostgreSQL).
